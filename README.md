# Visual Studio Code configuration

## Symbolic links for using the configuration

### On Linux
```
ln -s $(pwd)/keybindings.json $HOME/.config/Code/User/keybindings.json
ln -s $(pwd)/settings.json $HOME/.config/Code/User/settings.json
```
